Profil du type de données: AS Mailbox MSS Datatype Profile

URL officielle: https://interop.esante.gouv.fr/ig/fhir/annuaire/StructureDefinition/as-mailbox-mss Version: 1.2.0-snapshot-2
Draft à partir de 2026-06-19 Nom computable: AsMailboxMSSProfile

Datatype profile créé à partir ContactPoint dans le cadre de l'Annuaire Santé pour décrire les boîtes aux lettres du service de messagerie sécurisée de santé (MSSanté) rattachées aux professionnels et aux structures.

Utilisations:

  • Utilise ce/t/te profil de type de données: AS Organization Profile, AS Practitioner Profile and AS PractitionerRole Profile

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
