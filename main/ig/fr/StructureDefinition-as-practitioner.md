# AS Practitioner Profile - Annuaire Santé v1.2.0-snapshot-2

## Profil de ressource: AS Practitioner Profile 

 
Profil générique créé à partir de FrPractitioner dans le contexte de l'Annuaire Santé pour décrire les données liées à l'exercice professionnel d'un professionnel (professions à ordre, professions sans ordre, professionnels à rôle). 

**Utilisations:**

* Dérivé de ce Profil: [AS Donnée Publique Practitioner Profile](StructureDefinition-as-dp-practitioner.md) and [AS Donnée Restreinte Practitioner Profile](StructureDefinition-as-dr-practitioner.md)
* Référence ce Profil: [AS Person Profile](StructureDefinition-as-person.md) and [AS PractitionerRole Profile](StructureDefinition-as-practitionerrole.md)

Vous pouvez également vérifier [les usages dans le FHIR IG Statistics](https://packages2.fhir.org/xig/ans.fhir.fr.annuaire|current/StructureDefinition/as-practitioner)
