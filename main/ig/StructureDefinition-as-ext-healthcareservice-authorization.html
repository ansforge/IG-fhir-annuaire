Extension: AS HealthcareService Autorization Extension

Official URL: https://interop.esante.gouv.fr/ig/fhir/annuaire/StructureDefinition/as-ext-healthcareservice-authorization Version: 1.0.0-ballot-3
Active as of 2024-03-12 Computable Name: AsHealthcareServiceAuthorizationExtension

Extension créée dans le cadre de l’Annuaire Santé pour décrire les autorisations des activités sanitaires, sociales, médico-sociales et d’enseignement et des équipements matériels lourds autorisés.

Context of Use

This extension may be used on the following element(s):

  • Element ID HealthcareService

Usage info

Usage:

  • Use this Extension: AS Device Profile, AS HealthcareService HealthCare Activity Profile and AS HealthcareService Social Equipment Profile
