Resource Profile: As Device Profile

Official URL: http://interop.esante.gouv.fr/ig/fhir/annuaire-donnee-publique/StructureDefinition/as-device Version: 0.0.3
Active as of 2023-05-09 Computable Name: AsDeviceProfile

Profil créé à partir de la ressource Device dans le contexte de l’Annuaire Santé pour décrire les équipements matériels lourds (‘EML’) mis en oeuvre au sein d’un établissement.

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
