Resource Profile: AS PractitionerRole Profile

Official URL: http://interop.esante.gouv.fr/ig/fhir/annuaire/StructureDefinition/as-practitionerrole Version: 1.0.0-ballot
Active as of 2023-05-30 Computable Name: AsPractitionerRoleProfile

Profil créé à partir de FrPractitionerRoleExercice dans le contexte de l’Annuaire Santé pour décrire l’exercice professionel et la situation d’exercice contient les informations décrivant notamment la profession exercée, l’identité d’exercice d’un professionnel, le cadre de son exercice (civil, militaire, etc.) ainsi que les caractéristiques de l’exercice d’un professionnel pendant une période déterminée et dans une structure déterminée.

Usage:

  • Derived from this Resource Profile: AS Donnée Publique PractitionerRole Profile and AS Donnée Restreinte PractitionerRole Profile
