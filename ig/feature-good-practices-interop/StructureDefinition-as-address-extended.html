Data Type Profile: As Address Extended Profile

Official URL: http://interop.esante.gouv.fr/ig/fhir/annuaire-donnee-publique/StructureDefinition/as-address-extended Version: 0.0.3
Active as of 2023-04-13 Computable Name: AsAddressExtendedProfile

Extension créée dans le contexte de l’Annuaire Santé pour prise en compte des spécificités de l’adresse postale française

Usage:

  • Use this DataType Constraint: As Location Profile, As Organization Profile and AS Practitioner Profile
